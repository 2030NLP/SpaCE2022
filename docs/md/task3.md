



##### 2.1.3  子任务3：中文空间实体识别与空间方位关系标注任务

子任务3（中文空间实体识别与空间方位关系标注任务）的数据包含以下内容：

- ① qid：试题编号；
- ② context：存在空间语义异常的文本；
- ③ outputs：若干描述空间实体及其空间方位信息构成的18元组，描述了“某空间实体在某时，经由某事件，满足某种空间方位关系，这一命题的事实性为真/假”的信息。其中的元素有“文本片段”和“标签”两种类型。详细介绍请看下表。
- ④ corefs：列出了文中不同形但同指的文本片段。（**仅供评分使用，无需参赛者提交**）
- ⑤ non_corefs：列出了文中同形但不同指的文本片段。（**仅供评分使用，无需参赛者提交**）

元组中各个元素的含义及特点如下表所示：

|语义类型|序号|名称|数据类型|含义|约束|
|-------|---|---|-------|---|---|
| 非空间方位元素 | `[0]` | 空间实体 | 文本片段 | 对应于被描述空间方位的空间实体。 | 此元素是整个元组的核心，必定存在，不会为空（其余元素皆可能为空，即 `null`）。 |
| 非空间方位元素 | `[1]` | 空间实体2 | 文本片段 | 对应于与 `[0]` 号元素形成距离关系的另一个空间实体。 | 必定且仅与描述距离的 `[16]` 或 `[17]` 号元素之一同时出现，否则必为 `null`。其与 `[0]` 号元素应按文中出现顺序排序。 |
| 非空间方位元素 | `[2]` | 事件 | 文本片段 | 与空间实体的空间方位直接关联的事件。 |   |
| 非空间方位元素 | `[3]` | 事实性 | 标签 | 如果元组所述的空间方位命题是假的，则该字段为 `"假"`（否则为 `null`）。 |   |
| 非空间方位元素 | `[4]` | 时间（文本） | 文本片段 | 描述了空间实体处于某种空间方位关系的时间。 | 不会与 `[5]` 或 `[6]` 同时出现。 |
| 非空间方位元素 | `[5]` | 时间（标签）的参照事件 | 文本片段 | 如果空间实体处于某种空间方位关系的时间在文中并未写明但可以判断，则可通过此元素和 `[6]` 号元素共同描述。此元素描述了 `[6]` 号元素所参照的事件。 | 不会与 `[4]` 同时出现。必定伴随 `[6]` 出现。 |
| 非空间方位元素 | `[6]` | 时间（标签） | 标签 | 当文中未出现描述空间方位关系的具体时间，且该时间可以判断时，通过此元素描述空间方位关系的时间。 | 当 `[5]` 不为空，此元素可能的值为 `"说话时" / "过去" / "将来"` 。当 `[5]` 为空，此元素可能的值为 `"之时" / "之前" / "之后" / null` 。不会与 `[4]` 同时出现。 |
| 空间方位元素 | `[7]` | 处所 | 文本片段 | 描述静态空间实体相对某外部参照物的位置。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[8]` | 起点 | 文本片段 | 描述动态空间实体的方位发生变化的场景下，变化开始时实体的处所。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[9]` | 终点 | 文本片段 | 描述动态空间实体的方位发生变化的场景下，变化结束时实体的处所。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[10]` | 方向 | 文本片段 | 描述动态空间实体的位移方向。（空间实体在动态中才有方向特征） | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[11]` | 朝向 | 文本片段 | 描述空间实体某一侧面所朝向的位置。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[12]` | 部件处所 | 文本片段 | 描述空间实体作为一个部件在整体中的位置。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 非空间方位元素 | `[13]` | 部位 | 文本片段 | 描述了空间实体的某个部位。此元素出现时，当前的18元组描述的是此部件的位置而非整个空间实体的位置。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[14]` | 形状 | 文本片段 | 描述了空间实体的形状。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[15]` | 路径 | 文本片段 | 描述了空间实体位移时经过的轨迹。 | 不会与 `[16]` 或 `[17]` 同时出现。 |
| 空间方位元素 | `[16]` | 距离（文本） | 文本片段 | 描述了 `[0]` 和 `[1]` 之间的距离关系。 | 不会与其他空间方位元素（含 `[17]` ）同时出现。 |
| 空间方位元素 | `[17]` | 距离（标签） | 标签 | 描述了 `[0]` 和 `[1]` 之间的距离关系，可能的值为 `"远" / "近" / "变远" / "变近"`。 | 不会与其他空间方位元素（含 `[16]` ）同时出现。 |

<mark>更详细的数据说明将于子任务3正式发布时公布</mark>。
